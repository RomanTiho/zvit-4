# FootballHub - Веб-платформа для організації аматорських турнірів з футболу

## 📋 Опис проекту

FootballHub - це сучасна веб-платформа для організації та управління аматорськими футбольними турнірами. Платформа дозволяє створювати турніри, реєструвати команди, відстежувати результати матчів та переглядати турнірні таблиці.

## ✨ Основні можливості

- **Створення турнірів** - легке створення турнірів різних форматів (ліга, плей-офф, змішаний)
- **Реєстрація команд** - швидка реєстрація команд з інформацією про капітана та склад
- **Турнірні таблиці** - автоматичний розрахунок очок, голів та позицій команд
- **Розклад матчів** - перегляд майбутніх та завершених матчів
- **Фільтрація** - пошук турнірів за статусом, форматом та назвою
- **Адаптивний дизайн** - коректне відображення на всіх пристроях

## 🚀 Технології

- **HTML5** - семантична розмітка
- **CSS3** - сучасні стилі з градієнтами, анімаціями та адаптивним дизайном
- **JavaScript (Vanilla)** - динамічний функціонал без залежностей
- **LocalStorage** - збереження даних локально в браузері
- **Google Fonts (Inter)** - сучасна типографіка

## 💻 Як запустити

1. Клонуйте або завантажте проект
2. Відкрийте файл `index.html` у веб-браузері
3. Або використовуйте локальний сервер:
   ```bash
   # За допомогою Python
   python -m http.server 8000
   
   # За допомогою Node.js (npx)
   npx serve
   ```
4. Відкрийте браузер за адресою `http://localhost:8000`

## 📖 Використання

### Створення турніру
1. Натисніть кнопку "Створити турнір" на головній сторінці або сторінці турнірів
2. Заповніть форму з інформацією про турнір
3. Натисніть "Створити турнір"

### Реєстрація команди
1. Перейдіть на сторінку деталей турніру
2. Натисніть кнопку "Зареєструвати команду"
3. Заповніть форму з інформацією про команду
4. Натисніть "Зареєструвати команду"

### Перегляд результатів
1. Відкрийте сторінку деталей турніру
2. Перейдіть на вкладку "Таблиця" для перегляду турнірної таблиці
3. Перейдіть на вкладку "Матчі" для перегляду розкладу матчів

## 📊 Дані

Проект включає демонстраційні дані:
- 3 попередньо створених турніри
- 6 команд в активному турнірі
- Турнірна таблиця з результатами
- Розклад матчів

Всі дані зберігаються в LocalStorage браузера.
