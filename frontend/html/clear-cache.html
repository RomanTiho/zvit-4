🧹 Очищення кешу

Натисніть кнопку нижче, щоб видалити старі дані автентифікації адміністратора з localStorage та sessionStorage.

✅ Кеш успішно очищено!
Після очищення:
1. Закрийте всі вкладки з FootballHub
2. Натисніть Ctrl+Shift+R для жорсткого оновлення
3. Відкрийте admin.html та увійдіть знову
4. Відкрийте tournaments.html в новій вкладці
5. Оновіть сторінку - адмін-права не повинні з'явитися
